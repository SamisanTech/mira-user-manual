MIRA3D (Mesh Improvement Repair and Arrangement) 
performs a set of tasks on a 3D mesh  to make it "3D Printable". 
This software can be used to perform operations such as 
mesh repair, support generation, packing, slicing and file compression.

Key Features
++++++++++++

#. Automatic and Manual Mesh Repair.
#. Hole Filling. 
#. Mesh Extrusion of 3D Scanned Files/Shell to Part.
#. Hollow and Lattice Infill. 
#. Mesh Smoothing and Triangle Reduction.
#. Automatic and Manual Support Generation. 
#. 2D Nesting. 
#. SLC to STL conversion.
#. Raster Image Generation (with Anti-Aliasing/Grayscale).
#. File Compression and Export (without triangle reduction).
